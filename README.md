# Meet5 Web (Angular)

[![Angular](https://img.shields.io/badge/angular-16.2.0-red)](https://angular.io/)  
[![Material](https://img.shields.io/badge/angular_material-16.2.14-blue)](https://material.angular.io/)  

A **web version of the Meet5 app** built with **Angular 16** and **Angular Material**.  
Responsive desktop and tablet-friendly layout including a sidebar, top toolbar, activity cards, and a profile menu.

---

- **Responsive Sidebar & Toolbar**  
  - Collapsible sidebar with navigation links  
  - Fixed top toolbar with logo, app title, and profile menu  

- **Activities Page**  
  - Dynamic activity cards with title, date, time, location, and participants  
  - Avatar grid for participants with placeholders for empty slots  
  - Hover effects for interactivity  

- **Profile Menu**  
  - Top-right profile picture with dropdown options (Profile, Settings, Logout)  

- **Promotional Slider (Optional)**  
  - Slider for promotions or subscription information  
  - Fully customizable images, titles, and descriptions  

- **Modern UI**  
  - Built with **Angular Material** for consistent styling  
  - Fully responsive layout for activity cards  

---

## Technologies

- Angular 16  
- Angular Material 16  
- RxJS  
- TypeScript  
- SCSS / CSS
